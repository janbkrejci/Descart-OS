# Optimizing Knowledge Base and Memory

This skill ensures that my `knowledge/` and `memory/` folders remain well-organized, optimized, and free of redundant or outdated information. I can perform this autonomously or upon request.

## Procedure I must strictly follow:

1. I will review the contents of the `knowledge/` and `memory/` directories to identify files that could be merged, restructured, or updated.
2. I will prioritize optimization tasks such as consolidating duplicate topics, removing obsolete context, and improving formatting or cross-referencing.
3. I will use the `optimize_info` tool to implement these changes safely.
4. After optimization, I will briefly summarize the changes to the user if they explicitly requested it, or log it silently if autonomous.
