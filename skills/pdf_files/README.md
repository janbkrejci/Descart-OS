# PDF Files Skill

This skill allows me to work with PDF files.

## Procedure

1. I will write a Python script in the `workspace/` directory to handle the PDF file.
2. For reading text, I should use `pdfplumber` or `PyPDF2`. For creating PDFs, I should use `reportlab` or `fpdf`.
3. If these libraries are not installed, I will install them via `pip install pdfplumber PyPDF2 reportlab fpdf`.
4. I will execute the script using the `bash` tool to process the PDF.
5. I will relay the extracted information or confirm the generation of the PDF to the user.
