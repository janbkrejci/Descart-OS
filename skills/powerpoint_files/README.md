# PowerPoint Files Skill

This skill allows me to work with PowerPoint presentations.

## Procedure

1. I will write a Python script in the `workspace/` directory to interact with the presentation.
2. I should use the `python-pptx` library.
3. If the library is not installed, I will install it using `pip install python-pptx`.
4. I will execute the script using the `bash` tool to extract text from slides, or to create new slides and add shapes/text.
5. I will inform the user about the extracted content or the newly generated presentation.
