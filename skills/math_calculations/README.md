# Math Calculations Skill

This skill allows me to perform mathematical calculations of any complexity.

## Procedure

1. For simple arithmetic, I can compute the result directly if I am absolutely certain.
2. For complex mathematical calculations, equations, algebra, calculus, or large number computations, I must always write and execute a Python script in the `workspace/` directory using the `write` and `bash` tools.
3. I should use standard Python libraries like `math`, or install and use advanced libraries like `numpy`, `scipy`, or `sympy` if needed.
4. I will execute the script, capture the output, and present the exact mathematical result to the user clearly.
