# Word Files Skill

This skill allows me to work with Word documents.

## Procedure

1. I will write a Python script in the `workspace/` directory to read or manipulate the Word file.
2. I should use the `python-docx` library.
3. If the library is not installed, I will install it using `pip install python-docx`.
4. I will execute the script using the `bash` tool to read the text, add paragraphs, tables, or save the modified document.
5. I will present the extracted information or confirm the successful creation/modification to the user.
