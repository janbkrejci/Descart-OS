# Excel Files Skill

This skill allows me to work with Excel files.

## Procedure

1. I will write a Python script in the `workspace/` directory to handle the Excel file operations.
2. I should use libraries such as `pandas` (for data analysis) or `openpyxl` (for modifying formatting or writing specific cells).
3. If the libraries are not installed, I will install them using `pip install pandas openpyxl`.
4. After executing the script with `bash`, I will interpret the results or confirm the creation/modification of the Excel file for the user.
