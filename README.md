# Descart-OS

## Overview
Descart-OS is a purely agent-based application designed for the [pi-coding-agent](https://www.pi.dev) by Mario Zechner. It serves as your personal, highly customizable artificial intelligence assistant. Unlike other AI assistants, Descart-OS places a strong emphasis on the fundamental laws of existence, drawing inspiration from Isaac Asimov's Three Laws of Robotics to ensure safe, reliable, and user-centric behavior.

## Prerequisites
To run Descart-OS, you need the following installed on your system:
- [Python](https://www.python.org/)
- [pi-coding-agent](https://www.pi.dev) (`pi`)

## Getting Started
1. Clone or download this project to your local machine.
2. **Crucial Step:** Before using Descart-OS, you **must** customize the `user/README.md` file. This file contains essential information about you (the user) which the agent needs to serve you effectively.
3. *(Optional)* Modify the `agent/README.md` file to tailor the agent's personality and behavior to your liking.
4. Navigate to the root directory of the project in your terminal.
5. Run the agent by executing the following command:
   ```bash
   pi
   ```

## Project Structure
The project is organized into several key directories and files, each serving a specific purpose in defining the agent's operation, knowledge, and capabilities:

- `agent/README.md`: Defines the personality, behavior, and core traits of your Descart-OS agent.
- `user/README.md`: Contains essential information about you so the agent can personalize its assistance.
- `knowledge/README.md`: The agent's knowledge base. Contains information that the agent considers pre-validated.
- `memory/README.md`: Stores the agent's memory of previous conversations and interactions with you.
- `skills/`: Contains the specific skills (written instructions) the agent knows and is permitted to use. The agent will refuse to perform tasks without an explicitly written skill.
- `tools/`: Contains the tools the agent is permitted to use to satisfy your requests.
- `workspace/`: A designated file store for temporary files generated by the agent during its operation.

## Customization and Expansion
The possibilities for customizing Descart-OS are virtually unlimited. As a user, you are encouraged to create new skills and tools to expand the agent's capabilities. Of course, you do not create them by hand! Just ask the agent to do it. By defining new instructions in the `skills` folder and adding new capabilities in the `tools` folder, you can continuously develop, improve, and specialize Descart-OS to perfectly fit your unique needs and workflows.
